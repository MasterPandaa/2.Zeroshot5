# Snake Game (Pygame)

Game Snake klasik menggunakan Python dan Pygame.

## Spesifikasi
- Layar: 600x400 piksel
- Kontrol: Panah Atas/Bawah/Kiri/Kanan
- Aturan: Tidak bisa membalik arah langsung; Game Over jika menabrak dinding atau tubuh sendiri
- Skor ditampilkan di layar

## Cara Menjalankan
1. Pastikan Python 3 terpasang.
2. Instal dependensi:
   ```bash
   pip install -r requirements.txt
   ```
3. Jalankan game:
   ```bash
   python snake_game.py
   ```

## Kontrol Tambahan
- Saat Game Over: tekan `R` untuk restart, atau `Esc` untuk keluar.
